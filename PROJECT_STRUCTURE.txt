```

   └─ Data loading and validation

🔄 src/preprocessing.py                 [180 lines]
   └─ Data preprocessing pipeline
      Functions:
      - remove_identifier_columns(X, id_columns)
      - handle_categorical_features(X): One-hot encoding
      - ensure_numeric(X): Convert to float
      - prepare_features_and_target(df): Extract X, y
      - train_test_split_stratified(...): Stratified split
      - impute_missing_values(...): MICE or mean imputation
      - scale_features(...): StandardScaler or RobustScaler
      - save_preprocessing_objects(...): Save to disk
      - load_preprocessing_objects(...): Load from disk

🤖 src/train_models.py                  [180 lines]
   └─ Model training module
      Functions:
      - train_logistic_regression(...)

      - train_gradient_boosting(...)
      - train_ann_mlp(...)
      - train_all_models(...): Train all 4 at once
      - get_predictions(...): Generate probabilities
      - save_models(models, output_dir)
      - load_models(model_names, input_dir)
      - load_single_model(model_name, input_dir)

📊 src/evaluate.py                      [150 lines]
   └─ Model evaluation and metrics
      Functions:
      - compute_metrics(y_true, y_pred_proba): 6 metrics
      - evaluate_all_models(...): All models together
      - get_roc_data(predictions, y_test)
      - get_confusion_matrices(...)
      - get_detailed_report(...)
      - print_metrics_summary(results_df)
      - get_best_model(results_df, metric)
      - save_metrics(results_df, output_dir)

🔍 src/feature_importance.py            [200 lines]

      - extract_lr_coefficients(model, feature_names)

      - extract_all_feature_importance(...): All models
      - get_top_n_features(all_importance, n)
      - plot_feature_importance(all_importance, top_n)
      - save_feature_importance(...)

🎯 src/app.py                           [230 lines]
   └─ Main orchestrator - calls all modules
      11 steps:
      1. Load data
      2. Prepare features and target
      3. Train-test split (stratified)
      4. Imputation (MICE or mean)
      5. Feature scaling (StandardScaler)
      6. Train models (4 algorithms)
      7. Generate predictions (probability)
      8. Evaluate models (6 metrics)
      9. Plot ROC curves (PNG + PDF)
      10. Feature importance analysis
      11. Summary and report
```

## 🎯 What Each File Does

### If you want to...

**Train the model:**
- Use: `python src/app.py` (modular, recommended)
- Or: `python full_pipeline.py` (monolithic)

**Make predictions:**
- Use: `streamlit run streamlit_app.py`
- Requires: Models trained first

**Adjust hyperparameters:**
- Edit: `src/config.py`
- Then: Re-run training

**Add new features:**
- Edit: `src/preprocessing.py`
- Then: Re-run training

**Try different models:**
- Edit: `src/train_models.py` or `full_pipeline.py`
- Then: Re-run training

**Export metrics:**
- Check: `outputs/tables/model_metrics.csv`
- Or: `outputs/tables/model_metrics.xlsx`

**View feature importance:**
- Check: `outputs/tables/top_15_features.csv`
- Or: `outputs/figures/feature_importance.png`

**Load a trained model:**
- Use: `src/train_models.py` → `load_single_model()`
- Requires: Model pickle file
